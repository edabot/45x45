# 45x45

A massive grouping game that is a UI update to  "2025" by Thomas Colthurst (https://thomaswc.com/2025.html)

Click on pairs of tiles to combine into groups.
Click a tile and a group to add tile to a group.
Click on pairs of groups to join them.

Groups can be edited by clicking on edit icon on hover. Can change name and background color.

Groups can also be dragged to reorder them.

If you want to make your own variation of this with your own words, update the categories.js file. Just make sure that no words are used in more than one category.
